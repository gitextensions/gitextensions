Fugue Icons
===========

Many icons taken from the "fugue" icon set by Yusuke Kamiyamane.
Licensed under a Creative Commons Attribution 3.0 License.

http://p.yusukekamiyamane.com/
https://github.com/yusukekamiyamane/fugue-icons

We are using the shadowless versions of the icons.

It is possible to manually combine elements from within
the set to create new icons.

Gimp's "Colourise" feature can be used to rotate hues and
obtain different coloured versions in many cases.

See the full set of available icons here:

https://raw.githubusercontent.com/yusukekamiyamane/fugue-icons/master/all-shadowless.png
